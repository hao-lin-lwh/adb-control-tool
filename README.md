# Android 自动化助手

一个基于 PyQt5 的 Android 设备自动化控制工具，通过 ADB 命令实现设备控制、脚本执行等功能。

## 功能特性

### 设备管理
- **自动检测设备**：启动时自动检测连接的 Android 设备
- **设备选择**：支持多设备切换
- **实时状态显示**：显示设备连接状态

### 脚本编辑器
- **命令编辑**：支持手动输入 ADB 命令
- **双击编辑**：双击命令可快速编辑
- **脚本保存/加载**：支持脚本文件的保存和读取

### 按键模拟
- 电源键、音量+、音量-
- Home 键、返回键、菜单键
- 确认键、删除键

### 触摸模拟
- 单点点击
- 长按（1秒）
- 上滑、下滑、左滑、右滑

### 系统控制
- 截屏（保存到 /sdcard/screen.png）
- 录屏（保存到 /sdcard/record.mp4）
- 重启设备
- 关机
- 进入 Fastboot 模式

### 应用控制
- 启动浏览器、设置等系统应用
- 列出所有已安装应用
- 清除应用缓存

### 高级功能
- **命令延时**：可自动在命令间插入 1 秒延时
- **批量执行**：支持一次性执行多条命令
- **执行统计**：显示成功/失败命令数量

## 安装要求

### 系统要求
- Windows 10/11
- Python 3.6+

### 依赖库
```bash
pip install PyQt5
```

### ADB 工具
- 下载并安装 Android SDK Platform Tools
- 将 `platform-tools` 目录添加到系统 PATH
- 确保 ADB 可用：`adb devices` 命令能正常工作

## 使用方法

### 1. 连接设备
- 使用 USB 连接 Android 设备
- 在设备上启用「开发者选项」和「USB 调试」
- 确认设备已连接：`adb devices`

### 2. 选择设备
- 在程序顶部下拉框中选择设备
- 状态栏会显示连接状态

### 3. 执行命令
#### 方法一：使用快捷按钮
- 点击右侧面板的快捷按钮，自动插入 ADB 命令到编辑器
- 选中「所有命令自动插入1秒延时」可添加延时

#### 方法二：手动编辑脚本
- 在编辑器中直接输入 ADB 命令
- 格式：`adb -s <设备序列号> shell <命令>`

### 1. 保存和加载脚本
- 点击「保存脚本」将脚本保存到文件
- 点击「加载脚本」读取已保存的脚本

### 2. 执行脚本
- 点击「▶ 执行脚本」按钮
- 程序会逐行执行脚本中的命令
- 显示执行结果统计

## 命令示例

### 基础按键
```
adb -s <设备ID> shell input keyevent 26    # 电源键
adb -s <设备ID> shell input keyevent 3    # Home 键
```

### 触摸操作
```
adb -s <设备ID> shell input tap 500 800    # 点击 (500, 800)
adb -s <设备ID> shell input swipe 500 800 500 300 1000  # 上滑
```

### 截屏
```
adb -s <设备ID> shell screencap /sdcard/screen.png
```

### 录屏
```
adb -s <设备ID> shell screenrecord /sdcard/record.mp4
```

### 启动应用
```
adb -s <设备ID> shell am start -a android.intent.action.VIEW -d https://www.baidu.com
```

## 注意事项

1. **设备连接**：确保设备已启用 USB 调试模式
2. **权限问题**：某些操作需要设备 root 权限
3. **脚本执行**：执行脚本时请确保设备处于解锁状态
4. **延时设置**：开启「自动延时」后，每个命令后会自动等待 1 秒
5. **ADB 环境变量**：确保 ADB 已正确安装并添加到系统 PATH

## 常见问题

### Q: 提示「未检测到设备」？
A: 检查设备是否已启用 USB 调试，或尝试重新连接设备。

### Q: 如何查看设备序列号？
A: 在命令行输入 `adb devices`，第一列即为设备序列号。

### Q: 脚本执行失败？
A: 检查命令语法是否正确，确保设备已连接且处于解锁状态。

### Q: 如何获取设备屏幕坐标？
A: 在设备上使用「开发者选项」中的「显示触摸操作」查看坐标。
